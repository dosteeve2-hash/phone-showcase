## ✨ À propos

**AURA Pro** est une landing page produit premium qui simule la présentation d'un smartphone haut de gamme — inspiré des meilleures campagnes Apple et Nothing Phone. Un exercice de design ultra-soigné : animations fluides, mockups 3D interactifs, et narration produit immersive.

Ce projet démontre la maîtrise des animations web avancées avec Framer Motion et la création de landing pages produit dignes des grandes marques tech.

---

## 🎯 Sections

- **Hero** — Introduction épique avec animation d'entrée et tagline percutante
- **Intro** — Présentation du produit avec scroll-triggered reveal
- **Features** — Fonctionnalités clés avec icônes animées et micro-interactions
- **Phone Mockup** — Rendu 3D interactif du téléphone avec rotation au scroll
- **Color Gallery** — Sélecteur de coloris avec aperçu en temps réel
- **Specs** — Fiche technique avec animations staggered
- **CTA** — Section de conversion finale

## 🚀 Installation locale

```bash
# 1. Cloner le projet
git clone https://github.com/dosteeve2-hash/phone-showcase.git
cd phone-showcase

# 2. Installer les dépendances
npm install

# 3. Lancer en développement
npm run dev
# → http://localhost:3000
```

Aucune variable d'environnement requise.

## 🎨 Design

Ce projet s'inscrit dans la charte graphique SDC (dark mode by default) :
- Fond `#070e1f` → `#0c1528`
- Accent or `#f0a832` pour les highlights produit
- Animations soft, jamais agressives
- Scrollbar 3px or, sélection texte gold
